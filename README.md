# railsgirls-codespaces-sample

- [Rails Girls ガイド](https://railsgirls.jp/#guides-gides) を GitHub Codespaces 上で実施するためのベースプロジェクト

## はじめに

- ガイドは、日々、更新されています。常に最新のガイドを参照してください。

## 3. [Rails Girls インストール・レシピ](https://railsgirls.jp/install)

- [GitHub Codespaces を利用する](https://railsgirls.jp/install/codespaces)

### GitHub Codespaces を利用する

#### 1. ブラウザを確認する

#### 2. アカウントを作成する

- https://github.com/signup

#### 3. Codespaces のページに行く

- 代わりに [このページ](https://github.com/eitoball/railsgirls-codespaces-base) にアクセスしてください。

#### 4. Template をクリックする

- 代わりに "Use this template" をクリックしてください。

#### 5. Use this template をクリックする


"Open in a codespace" を選択してください。
- Fork したリポジトリのトップページから「Code」ボタンをクリック、「Codespaces」タブを選択し、「Create codespace on main」をクリックする。
    - 3〜5分かかかかります。

### 補足

- macOS や Windows OS 用セットアップのように以下のようにターミナルで以下のように Ruby と Rails のバージョンを確認できます。

```bash
$ ruby -v
ruby 4.0.1 (2026-01-13 revision e04267a14b) +PRISM [x86_64-linux]
```

```bash
$ rails -v
Rails 8.1.2
```
